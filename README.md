## Development

### Set up

Install deps with `asdf` and enable corepack (to use `pnpm`)

```sh
$ asdf install
$ corepack enable
$ pnpm install
```

### Start development server


```sh
$ pnpm run dev
```

### Build

```sh
$ pnpm run build
```
